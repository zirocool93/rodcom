# Инструкции для агентов

- Все интерфейсы, тексты, документацию и пользовательские сообщения по проекту писать на русском языке, если явно не указано иное.
- Вести документацию значимых изменений: что добавлено, как запустить, как проверить, какие ограничения остаются.
- Не коммитить секреты, реальные `.env`, дампы баз данных, пользовательские медиафайлы и резервные копии.
- Денежные суммы хранить в копейках в полях `amount_cents`, без float для расчетов.
- Для пользовательской финансовой лексики использовать мягкие формулировки: «ожидается оплата», «не поступило подтверждение», «остаток к оплате».
