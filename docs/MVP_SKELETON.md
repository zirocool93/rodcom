# MVP-скелет RodCom

Этот репозиторий сейчас содержит первую техническую базу проекта:

- Django backend с health endpoints и набором приложений под будущие доменные модули.
- Vue 3 PWA frontend с русским интерфейсом и заготовками ключевых страниц.
- aiogram 3 Telegram bot с `/start` и настройками SOCKS5 proxy через переменные окружения.
- Docker Compose для PostgreSQL, Redis, backend, frontend, Celery, bot и Nginx.
- Скрипты установки, обновления, резервного копирования и восстановления.

## Что еще не реализовано

- Доменные модели и миграции.
- Полные API для школ, семей, сборов, платежей, расходов и отчетов.
- Авторизация и разграничение прав.
- Генерация PDF/XLSX отчетов.
- Реальная привязка Telegram-аккаунтов и загрузка подтверждений оплаты.

## Следующий этап

Реализовать первую бизнес-цепочку:

`Класс -> Семьи -> Сбор -> Начисления -> Платеж -> Подтверждение -> Расход -> Баланс -> Отчет`.
